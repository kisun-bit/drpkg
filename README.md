# 二 介绍

`drpkg`是备份一体机使用的公共库，内部包含了文件、文件系统、数据处理、磁盘、网络、操作系统等等多方面的通用调用库，\
工程师可基于此快速构建备份一体机的主架构及主逻辑。

`drpkg`当前处于持续迭代过程中，笔者将依据工作经验，将更多场景下的通用调用集成至此库中。
